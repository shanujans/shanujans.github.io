# shanujans.github.io

This is a personal website repository built primarily with **TypeScript** (82.3%) and **HTML** (17.7%).

## Features

- Responsive site layout
- Clean TypeScript structure
- Easily customizable components

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/shanujans/shanujans.github.io.git
   ```

2. **Install dependencies:**  
   *(if applicable; for TypeScript projects using npm)*

   ```bash
   npm install
   ```

3. **Run locally:**  
   *(for static HTML, simply open `index.html` in your browser. For TypeScript, use build tools as needed.)*
